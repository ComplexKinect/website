The System

A distributed electrical and firmware system that allows for expandability and a compact, efficient mechanical design that minimizes parts for better functioning.

Computer Vision

Using OpenCV and a Raspberry Pi camera, our system tracks motion within the frame. Dividing the frame into quadrants corresponding to the physical apparatus of flowers, the system transmits which frame(s) have motion to the master arduino.

Mechanical Design

To minimize moving parts and the introduction of friction, our system uses a rack and pinion drive system and a brushless motor to move a line of 3 flowers. The brushless motor provides a powerful yet near silent method of driving a continuous drive gear which travels along the rack repeatedly to drive the pinion up and down with the flower attached.

Firmware & Electrical

In efforts to create a lightweight, scalable architecture, our electrical system features a leader-follower deployment of arduinos and an I2C communication bus to transmit data between them. The leader, or head node, receives serial commands from the Raspberry Pi, processes which follower node needs to act, and transmits that message to the leader node. The leader node then drives the motors to move the correct line according to the message it received.

Web App

To add an extra layer of interactivity, our system features a web app that allows the user to select which sensing type they want to use and how they want the system to respond. We give them the option of using audio, vision, or both to gather sensory data and give them the option to turn on and off light response as another output source.
